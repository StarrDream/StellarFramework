# 实用工具集 (Tools)

框架中包含了一系列“小而美”的工具脚本，拖拽即用。

| 脚本名 | 功能描述 | 使用场景 |
| :--- | :--- | :--- |
| **UniversalBillboard** | 通用公告板 | 血条、头顶名字，始终朝向摄像机。支持锁定 Y 轴。 |
| **FollowTarget** | 平滑跟随 | 摄像机跟随、宠物跟随。支持位置/旋转/缩放独立控制。 |
| **BoundsCalculator** | 边界计算器 | 自动计算复杂模型的包围盒大小（包含子物体）。 |
| **CameraScreenshot** | 截图工具 | 按 S 键截图，自动保存到项目根目录。 |
| **MiniFPS** | 简易 FPS 控制器 | 快速搭建测试场景，WASD 移动，右键旋转视角。 |
| **DynamicScaleLimiter** | 动态缩放限制 | 编辑器工具，限制物体缩放并自动吸附地面，适合摆放场景。 |
| **CombinedMeshCollider** | 网格合并碰撞 | 将复杂的子物体 Mesh 合并为一个 MeshCollider，优化物理性能。 |
| **UIDragger** | UI 拖拽 | 挂在 UI 上即可实现拖拽，支持限制在父节点范围内。 |
| **UIInputTrigger** | UI 事件触发器 | 支持双击、长按、悬浮事件，弥补原生 Button 的不足。 |

**使用建议：**
这些工具大多设计为 `[ExecuteAlways]` 或编辑器辅助，建议在开发阶段大量使用以提高效率。

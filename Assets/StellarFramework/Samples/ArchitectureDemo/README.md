# ArchitectureDemo / 架构样例

`Assets/StellarFramework/Samples/ArchitectureDemo` 用来存放完整业务链路示例。

## 目录

- `Scene/`
  演示场景。
- `Runtime/`
  配套运行时代码。
- `Resources/`
  案例依赖资源。

## 主要入口

- `Scene/Demo.unity`
  推荐优先打开的主场景。
- `Scene/FrameworkArchitecture_DemoScene.unity`
  架构说明场景。
- `Scene/FrameworkArchitecture_Playable.unity`
  可直接运行的架构样例场景。

## 建议阅读顺序

1. 打开 `Scene/Demo.unity`
2. 查看 `Runtime/DemoEntry.cs`
3. 查看 `Runtime/Architecture/`
4. 再回到 `Samples/KitSamples`

如果想先建立整体理解，从这里开始最直接。
